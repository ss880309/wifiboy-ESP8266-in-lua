# wifiboy-ESP8266-in-lua　　
2016年剛問世時，我們還是以lua撰寫開發版寫入程式，如今IDE統一之後，僅需要透過標準語言即可。  
wifiboy為老師與原創開發者合作結合課堂教材，使用初代未上市的測試版進行開發。  
此課程可視為設計數學的延伸課程。
授課老師：靜宜大學資訊傳播與工程學系 愛新覺羅燾昍(金城) 教授  
課程：互動設計系統概論
